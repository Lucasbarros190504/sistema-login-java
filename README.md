# Sistema de Login Java 🔐

Sistema de autenticação simples desenvolvido em Java como projeto de estudo.

## 📋 Funcionalidades

- ✅ Cadastro de usuário com confirmação de senha
- ✅ Login com validação de credenciais
- ✅ Limite de 3 tentativas de login
- ✅ Contador de tentativas restantes
- ✅ Menu interativo com opção de sair

## 🛠️ Tecnologias utilizadas

- Java
- Scanner (entrada de dados do usuário)

## 🚀 Como executar

1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe `SistemaLogin.java`
